# 漫画图片下载 - 需求分析

## 1. 数据流

```
comics_metadata.json (734部)
  └─ 遍历每部漫画
       ├─ 已有: comics_detail/{编号}_{标题}/cover.jpg
       ├─ 已有: comics_detail/{编号}_{标题}/metadata.json
       │
       ├─ API: GET /comics/{comic_id}/eps → 章节列表
       │    └─ { docs: [{_id, title, order}, ...], total, pages }
       │
       └─ API: GET /comics/{comic_id}/order/{order}/pages?page=1..N → 图片列表
            └─ { pages: { docs: [{_id, media: {fileServer, path, originalName}}], pages } }
            └─ 图片URL = "{fileServer}/static/{path}"
```

## 2. 目录结构

```
comics_detail/
├── 001_浪客行/
│   ├── cover.jpg
│   ├── metadata.json
│   ├── chapters.json              ← 章节列表缓存
│   ├── 第01卷/
│   │   ├── 001.jpg
│   │   ├── 002.jpg
│   │   ├── ...
│   │   └── 244.jpg
│   ├── 第02卷/
│   │   └── ...
│   └── 第13卷/
│       └── ...
```

## 3. API 数据

### 3.1 章节列表

```
GET /comics/{id}/eps?page=1
→ data.eps.docs: [{_id, title, order, updated_at}, ...]
→ data.eps.total: 13
→ data.eps.pages: 1   (章节数多时会分页)
```

### 3.2 图片列表

```
GET /comics/{id}/order/{order}/pages?page=1
→ data.pages.docs: [{_id, media: {fileServer, path, originalName}}, ...]
→ data.pages.total: 244   (本章总页数)
→ data.pages.pages: 7     (API 分页数，每页 40 张)
→ data.ep: {_id, title}   (本章信息)
```

图片 URL 拼接：`{fileServer}/static/{path}`

### 3.3 实例数据

| 漫画 | 章节数 | 总页数 | 单话最大页数 |
|------|--------|--------|------------|
| 浪客行 | 13话 | 2770P | 244P (第1话) |
| 博士の夜の助手 | 5话 | 199P | ~40P |

## 4. 功能设计

### 4.1 下载模式

| 命令 | 说明 |
|------|------|
| `python manga_api.py download <编号>` | 下载指定编号漫画的全部章节 |
| `python manga_api.py download 1-50` | 下载编号范围 |
| `python manga_api.py download all` | 下载全部 734 部漫画的全部章节 |

### 4.2 断点续传

- **章节级**: 检查章节文件夹内图片数量是否 ≥ `total` 页数，跳过已完成章节
- **图片级**: 单个图片文件已存在且大小 > 0 则跳过
- **进度记录**: 每完成一个章节后更新 `chapters.json`，支持中断恢复

### 4.3 并发控制

- 章节之间串行下载（逐个请求 API）
- 单话内的图片可以并发下载（3-5 线程）
- 全局请求间隔：1.5 秒（API 请求）/ 0 秒（图片下载）

### 4.4 错误处理

- API 请求失败自动重试 3 次
- 图片下载失败记录到日志，跳过继续
- 中断后可重新运行，自动跳过已下载的

## 5. chapters.json 格式

```json
{
  "comicId": "67a18a90cac76a1659ab71f5",
  "chapters": [
    {
      "order": 1,
      "title": "第1卷",
      "totalPages": 244,
      "downloaded": false,
      "imageCount": 0
    }
  ],
  "lastUpdate": "2026-06-15T20:00:00"
}
```

## 6. 估算

| 项目 | 数值 |
|------|------|
| 总漫画数 | 734 部 |
| 平均每部章节数 | 3-5 话 |
| 平均每话页数 | 40P |
| 估算总图片数 | 约 10-15 万张 |
| 每张图片大小 | 100KB-500KB（medium 画质）|
| 估算总大小 | 10GB - 75GB |
| 全部下载耗时 | 估约 5-10 小时 |

## 7. 注意事项

- 图片服务器 `storage-b.picacomic.com` 需要 `Referer` 头
- `image-quality` 头控制画质（fast/medium/hd/original），影响文件大小
- 图片 URL 中的 `rs:fit:800:800:0` 是缩放参数，去掉可获取原图
- 单日 API 请求频率不能太高，建议章节间间隔 ≥ 1.5 秒
